Veil of Mortanox

A cursed 2D pixel-fantasy platformer forged with Pygame

About the Game

Veil of Mortanox is a dark-fantasy 2D pixel platformer centered around Mortanox — a cursed, heroic figure in a mythic land. Collect Lucarii (coins), protect your Vitalis (health), defeat hostile bats and mini-bosses, and uncover the mystery behind the curse.

The game currently features basic enemy logic, sprite-based animation, combat mechanics, and collectible Lucarii. A full upgrade shop, healing system, boss battles, and more are under development.

Current Gameplay Elements

  • Mortanox character with attack, jump, and run animations
  • Hostile bats with visible health bars
  • Mini-boss spawn system (every 5 bat kills)
  • Lucarii and Vitalis UI, coin collection logic
  • Level-up and save system (JSON-based)
