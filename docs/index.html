Veil of Mortanox

A dark fantasy 2D pixel platformer built with Python and Pygame

Gameplay Overview

Veil of Mortanox follows the cursed hero Mortanox as he fights dark creatures, collects Lucarii (coins), and defends his Vitalis (health) in a decaying world.

Core Features

  • Play as Mortanox – a Latin-inspired cursed hero with smooth pixel animations
  • Collect Lucarii – In-game currency for upgrades and healing
  • Vitalis System – Health points with visual feedback (5 hearts to start)
  • Combat – Sword attack animations with collision damage (no enemy AI yet)
  • Hostile Bats – Basic enemies dealing 1 Vitalis damage, taking 3 hits to defeat
  • Mini-Bosses – Under development; spawn every 5 bat kills with planned AI
  • Progression – Level increases after each mini-boss defeat (tracked in level.json)
  • Lucarii Saving – Coin saving between sessions using JSON files
  • Pixel Art & Sound – Original FL Studio music and Piskel/Photoshop pixel art (placeholders currently)

In Development

  • Enemy AI (starting with mini-boss)
  • Upgrade Shop
  • Final Backgrounds & Tiles
  • More Sound Effects & Music
  • "How to Play" Tutorial Screen
  • Boss Battle Logic

How to Run

  1. Ensure Python 3.x and Pygame are installed:
    pip install pygame
  2. Run the menu.py file to start the game
